# Rals UI

A simple react ui component library reminiscent of my favorite fictional character ralsei :3

## Installation

```bash
npm i rals-ui
```

## Usage

```tsx
import { Button, Container, Group } from "rals-ui";

export default function App() {
  return <Button>Click Me!</Button>;
}
```
